# TimeToRent (T2R)

A streamlined marketplace platform designed to bridge the trust gap between buyers and renters.

## Problem Statement
The current digital marketplace is oversaturated with fragmented services for renting and buying. This fragmentation leads to significant trust issues, unreliable listings, and excessive time wasted navigating multiple platforms. T2R centralizes these transactions into a secure, transparent environment.

## Features
* **User Authentication**: Secure account creation, login, and session management.
* **Listing Management**: Users can easily upload, edit, and manage their own publications and items.
* **Rating & Feedback System**: Integrated community trust through user ratings and detailed comments.
* **Flexible Transactions**: Integrated support for both rental agreements and direct purchases.
* **Geolocation**: Visualizing listings using interactive maps for local convenience.

## Technology Stack
* **Backend**: Java, Spring Boot
* **Frontend**: JavaScript, React, TailwindCSS
* **Database**: PostgreSQL
* **Cloud & Tools**: AWS, GitHub, Postman
* **APIs**: Google Maps API

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/adilkhanalimberdi/advancedproject
   ```

2. **Backend Setup**:
   * Navigate to `src/backend`.
   * Configure `application.properties` with your PostgreSQL credentials.
   * Run the application via IntelliJ IDEA or using `./mvnw spring-boot:run`.

3. **Frontend Setup**:
   * Navigate to `src/frontend`.
   * Install dependencies: `npm install`.
   * Start the development server: `npm start`.
